# Images Directory

Place your images here:

## Required Images

- **logo.png** - Company logo for header
- **og-image.jpg** - Open Graph image (1200x630px)
- **favicon-16x16.png** - 16x16 favicon
- **apple-touch-icon.png** - 180x180 Apple touch icon
- **android-chrome-192x192.png** - 192x192 Android icon
- **android-chrome-512x512.png** - 512x512 Android icon

## Case Study Images

Place case study images in this directory with the following naming convention:
- `case-studies/[case-study-id].jpg`

Example:
- `case-studies/defi-protocol.jpg`
- `case-studies/nft-marketplace.jpg`
- etc.

## Image Optimization Tips

- Use WebP format when possible
- Optimize images before uploading (use tools like TinyPNG, ImageOptim, etc.)
- Recommended dimensions:
  - OG images: 1200x630px
  - Case study images: 800x600px or 16:9 aspect ratio
- All images should be under 200KB for optimal performance
